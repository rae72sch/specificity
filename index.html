CSS Specificity

Colour coded by Priority


Name Selector Description Example
Important !important !Important is a rule which overrides ALL previous styling rules for that specific property on that element p {
background-color: red !important;
}
Inline Styles <tag style=""> Inline Styles have second priority <h1 style="color: blue">
ID #id ID has third priority, and selects the element with id="firstname" #firstname {}
Class .class Class has joint fourth priority, and selects all elements with class="intro" .intro {}
Pseudo-Class selector:pseudo-class Pseudo-Class has joint fourth priority, and selects paragraph elements when hovered over p:hover {}
Attribute [attribute] Attibute has joint fourth priority, and selects all elements with a href attribute [href] {}
Element element Element has joint fifth priority, and selects all instances of h1 h1 {}
Pseudo-Element selector::pseudo-element Pseudo-Element has joint fifth priority, and selects all first lines in all paragraph elements p::first-line {}
Universal Selector * The Universal Selector has last priority, and selects all elements * {}
